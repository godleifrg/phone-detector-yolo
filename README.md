# Phone Detector

Тестовое задание на позицию Computer Vision Engineer.

## Описание

Консольное приложение на Python для поиска мобильных телефонов на изображениях с использованием YOLOv8.

### Возможности

- поиск телефонов на изображении;
- обработка всех изображений из папки `images`;
- сохранение изображения с разметкой;
- сохранение каждого найденного телефона отдельным файлом;
- вывод информации в консоль.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Структура проекта

```text
images/
main.py
requirements.txt
```

При запуске проекта должна создаться папка **output**, в которую будут добавляться детектируемые файлы
